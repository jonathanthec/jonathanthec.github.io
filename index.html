My Projects (more coming soon)

Rent Tracker


Rent Tracker is a landlord's best friend. It helps the user to keep track of their existing rental properties, as well as tenants of those properties. It also keeps track of payment information, so the user will never forget about collecting rent again.


Rent Tracker's backend is built using Node JS, GraphQL and Prisma, while its frontend is built using React and Apollo.


AdaptivApps


AdaptivApps is created for Angel City Sports, a non-profit organization that hosts atheltic events for youth with disabilities. This app is created to ensure smooth communication among ACS staff, volunteers, athletes, and parents. It features a calender system that allows user to sign up for various events and activities, as well as a messaging system that enables smooth communication.


AdaptivApps's backend is built using GraphQL, Prisma, and Apollo client. Its its frontend is built using React and Apollo. The app uses auth0 for authentication and is deployed through AWS. It also features Google Analytics and a customized widget for website accessibility.


Pathfinding Visualizer


This is a pathfinding visualizer that is built using React JS, Bootstrap,
and plain CSS! It will be featuring multiple pathfinding and
maze-generation algorithms. More features coming soon!


All animation effects are built using CSS. Some components are incorporated from Bootstrap.


House Price Prediction Model


A data science/machine learning project for a Kaggle competition. My team and I used advanced regression techniques and machine learning methods to devise a model that predicts the prices for properties in Boston.


This project is written in Python. It employs advanced techniques, including LASSO, gradient boosting, and xgboost/cross validation.
